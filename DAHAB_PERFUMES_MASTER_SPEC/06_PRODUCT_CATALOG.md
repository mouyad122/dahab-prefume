## 4. Admin-Editable Fields Mapping
To allow changes to notes and details in the admin portal, the data store schema must define the following fields as editable:
* `title` (AR/EN object)
* `shortDescription` (AR/EN object)
* `longDescription` (AR/EN object)
* `price` (Number)
* `compareAtPrice` (Number, nullable)
* `stock` (Number)
* `fragranceNotes` (Object: `top` Array, `heart` Array, `base` Array)
* `metrics` (Object: `longevity` String, `sillage` String, `bestSeason` Array, `bestTime` String)
* `hidden` (Boolean)
* `featured` (Boolean)
* `newArrival` (Boolean)
* `bestseller` (Boolean)
